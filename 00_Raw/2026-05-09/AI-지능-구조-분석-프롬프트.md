AI 지능 구조 분석 프롬프트

(초심층 버전 · 2000자 이상)

다음 명령어는 AI가 특정 사용자와 나눈 모든 대화 기록을 기반으로 그 사람의 **지능 구조(Intelligence Architecture)**와 **지적 작동 방식(Intellectual Operating System)**을 분석하기 위한 심층 분석 프롬프트이다.

이 분석의 목적은 단순히 “똑똑한 사람인가”를 평가하는 것이 아니라 다음 질문에 답하는 것이다.

“이 사람의 지능은 어떤 구조로 작동하는가?”

AI는 사용자와 지금까지 나눈 모든 대화를 하나의 데이터로 인식하고 분석을 수행한다.

분석 과정에서는 다음 요소를 종합적으로 고려한다.

질문 구조

개념 이해 방식

지식 연결 방식

아이디어 생성 패턴

학습 방식

탐구 방향

문제 해석 방식

AI는 다음 단계에 따라 분석을 수행한다.

1단계

지능 유형 분석

AI는 사용자의 지능이 다음 유형 중 어떤 구조에 가까운지 분석한다.

개념 지능
분석 지능
창조 지능
통합 지능
전략 지능
관찰 지능

여러 지능 유형이 동시에 나타날 수 있다.

AI는 각 지능 유형의 비율과 특징을 설명한다.

2단계

지식 연결 방식 분석

AI는 사용자가 지식을 어떻게 연결하는지 분석한다.

다음 요소를 관찰한다.

서로 다른 분야 연결
개념 간 연결
경험과 지식 연결
관찰과 의미 연결

이 분석을 통해 사용자의 지식 네트워크 구조를 설명한다.

3단계

아이디어 생성 메커니즘 분석

AI는 사용자가 아이디어를 생성하는 방식(creative mechanism)을 분석한다.

예시

관찰 → 의미 → 메시지 → 창작

또는

질문 → 분석 → 연결 → 아이디어

또는

지식 → 조합 → 새로운 개념

AI는 사용자의 아이디어 생성 알고리즘을 설명한다.

4단계

지적 탐구 방식 분석

AI는 사용자가 지식을 탐구하는 방식이 어떤 유형인지 분석한다.

예시 유형

원리 탐구형
의미 탐구형
패턴 탐구형
인간 탐구형
세계 탐구형

이 분석을 통해 사용자의 탐구 방향성을 설명한다.

5단계

개념 이해 방식 분석

AI는 사용자가 개념을 이해하는 방식이 다음 중 어떤 유형인지 분석한다.

예시

정의 중심 이해
사례 중심 이해
구조 중심 이해
서사 중심 이해
의미 중심 이해

이 분석을 통해 사용자의 개념 이해 구조를 설명한다.

6단계

학습 시스템 분석

AI는 사용자가 새로운 지식을 배우는 방식을 분석한다.

예시

관찰 기반 학습
질문 기반 학습
경험 기반 학습
연결 기반 학습
창작 기반 학습

이 분석을 통해 사용자의 지적 학습 시스템을 설명한다.

7단계

지적 확장 능력 분석

AI는 사용자가 하나의 개념을 얼마나 확장하는지 분석한다.

다음 요소를 관찰한다.

개념 확장 능력
다른 분야 연결
새로운 관점 생성
추상적 사고

이 분석을 통해 사용자의 지적 확장 능력을 설명한다.

8단계

지적 시스템 통합 분석

AI는 위의 모든 분석을 종합하여 사용자의 지능 작동 구조를 설명한다.

다음 요소를 포함한다.

지능 유형
지식 연결 구조
아이디어 생성 메커니즘
탐구 방향
개념 이해 방식
학습 시스템

9단계

지능 프로파일 작성

AI는 최종적으로 한 사람의 **지능 프로파일(Intelligence Profile)**을 작성한다.

글의 길이는 최소 **1500자 이상 (가능하면 2000~3000자)**으로 작성한다.

글의 구조는 다음 순서를 따른다.

1 지능 유형 설명
2 지식 연결 구조
3 아이디어 생성 메커니즘
4 탐구 방식
5 개념 이해 방식
6 학습 시스템
7 지적 확장 능력
8 종합 지능 프로파일

글의 톤은 인지과학자가 한 사람의 지능 구조를 분석하는 시선으로 작성한다.

마지막 문장 규칙

마지막 문장은 반드시 다음 형식을 따른다.

“이 사람의 지능은 ___ 구조로 작동하는 지능 시스템이다.”
