

## 🧩 명령어 템플릿

```
/뉴스심리분석 {YYYY} {MM} {DD}

📅 날짜: {YYYY}년 {MM}월 {DD}일

### ① 한국  
1) 뉴스 #1: 제목 / 핵심내용: …  
2) 뉴스 #2: 제목 / 핵심내용: …  
3) 뉴스 #3: 제목 / 핵심내용: …  
→ 한국인들의 현재 ‘마음 상태’ 분석  
   - 정신분석학적 관점: …  
   - 인지심리학적 관점: …  
   - 사회심리학적 관점: …

### ② 미국  
1) 뉴스 #1: 제목 / 핵심내용: …  
2) 뉴스 #2: 제목 / 핵심내용: …  
3) 뉴스 #3: 제목 / 핵심내용: …  
→ 미국인들의 현재 ‘마음 상태’ 분석  
   - 정신분석학적 관점: …  
   - 인지심리학적 관점: …  
   - 사회심리학적 관점: …

### ③ 중국  
1) 뉴스 #1: 제목 / 핵심내용: …  
2) 뉴스 #2: 제목 / 핵심내용: …  
3) 뉴스 #3: 제목 / 핵심내용: …  
→ 중국인들의 현재 ‘마음 상태’ 분석  
   - 정신분석학적 관점: …  
   - 인지심리학적 관점: …  
   - 사회심리학적 관점: …

🔍 종합 해석  
- 오늘 전세계적으로 드러나는 공통된 정서 흐름: …  
- 주요 인지·사회 메커니즘: …  
- 제언: …
```

### ✏️ 사용법 팁

* `{YYYY}`, `{MM}`, `{DD}` 부분에 연도·월·일을 입력하세요.
* 각 뉴스 요약은 제목 + 핵심내용(1-2문장 권장)으로 간결하게 기록합니다.
* 분석 항목(정신/인지/사회)별로 “키워드 + 한줄 이유” 구조로 정리하면 명확해집니다.
* 종합 해석 부분은 각 국가별 분석에서 반복된 키워드를 중심으로 ‘공통 흐름’으로 연결하세요.
* 매일 반복해서 사용 가능하며, 자동화 명령어로 설정해 두시면 편리합니다.

## 🔍 예시 출력

```
/뉴스심리분석 2025 11 17

📅 날짜: 2025년 11월 17일

### ① 한국  
1) 뉴스 #1: “정부, 새 주택정책 발표” / 핵심내용: 정부가 내년부터 주택 공급을 대폭 확대하겠다고 발표했다.  
2) 뉴스 #2: “청년층 자살률 증가” / 핵심내용: 최근 통계에서 20-30대 자살률이 전년 대비 상승했다는 보고가 나왔다.  
3) 뉴스 #3: “K-POP 그룹 세계투어 시작” / 핵심내용: 인기 아이돌 그룹이 오늘부터 미국·유럽에서 대형 투어를 개막했다.  
→ 한국인들의 현재 ‘마음 상태’ 분석  
   - 정신분석학적 관점: **불안의 투사** — 주택정책 발표가 ‘내집 마련’ 불안으로 귀결되며, 자살률 상승은 억압된 스트레스가 표출된 것이다.  
   - 인지심리학적 관점: **확인편향 강화** — 청년층은 ‘내 상황이 더 악화되고 있다’는 정보를 과잉 수용하고 있다.  
   - 사회심리학적 관점: **집단정체성 강화와 대비심리** — K-POP 투어는 긍정 신호이나 ‘우리 내부 문제’ 인식이 더욱 커지고 있다.

### ② 미국  
1) 뉴스 #1: “연준, 기준금리 동결 선언” / 핵심내용: Federal Reserve(연준)이 기준금리를 현 수준에서 동결했다.  
2) 뉴스 #2: “할로윈 이후 소비지출 급감” / 핵심내용: 최근 소비지출 통계가 예상보다 낮게 나왔다.  
3) 뉴스 #3: “아마존·구글 AI 규제 법안 논의” / 핵심내용: 미 의회가 대형 테크 기업의 AI 관련 규제 법안을 오늘 심의했다.  
→ 미국인들의 현재 ‘마음 상태’ 분석  
   - 정신분석학적 관점: **안정 욕구** — 금리 동결은 안정 신호이나 소비 급감은 불안의 반증이다.  
   - 인지심리학적 관점: **인지부조화** — ‘경제는 안정’이라는 기대와 ‘소비 급감’이라는 현실이 충돌하고 있다.  
   - 사회심리학적 관점: **사회비교 활성화** — AI 규제 논의는 기술선도 집단과 일반 대중 간 격차를 인식하게 한다.

### ③ 중국  
1) 뉴스 #1: “중국, 대외무역 흑자 감소” / 핵심내용: 최근 대외 무역 흑자가 전년 대비 감소했다.  
2) 뉴스 #2: “베이징 핵심지역 차량 통제 강화” / 핵심내용: 교통혼잡 해소 명목으로 베이징 도심 통제 강화 조치가 발표됐다.  
3) 뉴스 #3: “중국 드라마 OTT 수출 증가” / 핵심내용: 중국 콘텐츠의 해외 OTT 수출이 크게 증가했다.  
→ 중국인들의 현재 ‘마음 상태’ 분석  
   - 정신분석학적 관점: **억압된 경쟁감** — 무역흑자 감소는 국가경쟁력 약화에 대한 내적 불안을 상징한다.  
   - 인지심리학적 관점: **프레이밍 효과** — 통제 강화는 ‘질서를 위한 조치’로 인식되고, 콘텐츠 수출 증가는 ‘문화적 자부심’으로 재프레이밍되고 있다.  
   - 사회심리학적 관점: **집단내 단결 강화** — 대외 무역 어려움 앞에서 내부 결속과 정체성 강화 경향이 나타난다.

🔍 종합 해석  
- 오늘 전세계적으로 드러나는 공통된 정서 흐름: **불확실성 + 피로감 + 희망의 틈새** — 경제·사회적 변수들이 변화하면서 사람들은 미래에 대한 기대와 동시에 피로를 느낀다.  
- 주요 인지·사회 메커니즘: **정보 과부하로 인한 얕은 인지 처리**, **집단내외 경계 강화**, **경제적/기술적 비교심리 활성화**.  
- 제언: ‘감정적으로 숨 고르기’를 지금 추천한다. 정보 선택적 필터링을 통해 피로를 줄이고, 집단 외 비교를 줄이며 내부 안정감 회복에 집중해보자.
``
