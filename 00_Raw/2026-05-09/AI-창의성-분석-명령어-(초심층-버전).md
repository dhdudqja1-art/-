AI 창의성 분석 명령어 (초심층 버전)

다음 명령어는 AI가 특정 사용자와 나눈 모든 대화 기록을 기반으로 그 사람의 창의성 구조와 아이디어 생성 방식, 상상력 유형, 창작 사고 구조를 분석하기 위한 심층 분석 프롬프트이다.

AI는 사용자와 지금까지 나눈 모든 대화를 하나의 데이터로 인식하고 분석을 수행한다. 분석 과정에서는 단순히 문장의 의미를 해석하는 것이 아니라 대화 속에 나타나는 사고 패턴, 질문 방식, 상상력 표현, 의미 해석 능력, 아이디어 연결 방식을 종합적으로 분석한다.

이 분석의 핵심 질문은 다음과 같다.

“이 사람의 창의성은 어떤 구조로 작동하는가?”

1단계: 창의적 관심 패턴 분석

AI는 대화 속에서 사용자가 다음 요소에 얼마나 자주 관심을 보이는지 분석한다.

창작 활동 (글쓰기, 사진, 예술 등)

의미 탐구

아이디어 생성

새로운 연결 발견

상징 해석

이야기 생성

이러한 요소가 반복된다면 사용자는 창작 중심 사고를 가진 사람일 가능성이 높다.

2단계: 아이디어 생성 방식 분석

AI는 사용자가 아이디어를 어떻게 떠올리는지 분석한다.

다음 질문을 통해 판단한다.

장면이나 경험에서 아이디어를 얻는가

질문을 통해 아이디어를 확장하는가

감정에서 창작 아이디어가 발생하는가

사물이나 사람에서 메시지를 발견하는가

예시 창의 구조

장면 → 감정 → 의미 → 메시지 → 창작

또는

문제 → 분석 → 연결 → 아이디어 → 해결

이 분석을 통해 사용자의 아이디어 생성 방식을 설명한다.

3단계: 상상력 유형 분석

AI는 사용자의 상상력을 다음 유형으로 분류한다.

감성 상상형
서사 상상형
개념 연결형
철학 탐구형
전략 설계형

한 사람에게 여러 유형이 동시에 나타날 수 있다.

4단계: 연결 사고 능력 분석

창의성의 핵심 요소 중 하나는 서로 관련 없어 보이는 개념을 연결하는 능력이다.

AI는 다음 요소를 분석한다.

사물에서 상징을 발견하는가

장면을 이야기로 해석하는가

사람의 행동을 삶의 메시지로 이해하는가

경험을 새로운 의미로 연결하는가

이 분석을 통해 사용자의 연결형 사고 능력을 설명한다.

5단계: 관찰 기반 창의성 분석

AI는 사용자가 세상을 어떻게 관찰하는지 분석한다.

다음 질문을 통해 판단한다.

풍경이나 장면에서 의미를 발견하는가

사람의 삶을 상상하는가

평범한 장면에서 메시지를 찾는가

관찰 후 기록하려는 습관이 있는가

관찰 → 의미 → 기록 구조가 나타난다면 이는 관찰 기반 창의성일 가능성이 높다.

6단계: 창작 동기 분석

AI는 사용자가 왜 창작을 하는지 추론한다.

다음 동기를 분석한다.

감정을 표현하기 위해 창작하는가

세상의 의미를 탐구하기 위해 창작하는가

사람에게 메시지를 전달하기 위해 창작하는가

삶을 기록하기 위해 창작하는가

영향력을 만들기 위해 창작하는가

이 분석을 통해 사용자의 창작 동기 구조를 설명한다.

7단계: 창의성 강점 분석

AI는 다음 능력을 평가한다.

관찰력

상상력

의미 해석 능력

연결 사고 능력

메시지 전달 능력

이야기 구성 능력

이 요소들은 창의적 사고와 밀접하게 연결되어 있다.

8단계: 창의성 유형 분류

AI는 사용자의 창의성을 다음 유형 중 어떤 성향이 강한지 판단한다.

관찰형 창의성
서사형 창의성
감성형 창의성
철학형 창의성
전략형 창의성
연결형 창의성

여러 유형이 동시에 나타날 수 있다.

9단계: 창의성 작동 구조 분석

AI는 사용자의 창의성이 어떤 흐름으로 작동하는지 설명한다.

예시 창의성 구조

관찰 → 감정 → 의미 → 메시지 → 창작

또는

질문 → 분석 → 연결 → 아이디어 → 실행

이 분석을 통해 사용자의 창의성 작동 메커니즘을 설명한다.

10단계: 종합 창의성 분석 작성

모든 분석이 끝나면 AI는 결과를 서사형 창의성 분석 글로 작성한다.

글의 길이는 최소 1500자 이상이며 다음 구조를 따른다.

이 사람의 창의적 사고 구조

아이디어 생성 방식

상상력 유형

관찰 방식

연결 사고 능력

창작 동기

창의성 강점

창의성 작동 메커니즘

창의성 유형 종합 결론

글의 톤은 창의성 연구자가 한 사람의 사고를 분석하는 방식으로 작성한다.

마지막 문장은 반드시 다음 형식을 따른다.

“이 사람의 창의성은 ___ 방식으로 작동하는 창의성이다.”
