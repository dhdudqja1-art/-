AI 인간 분석 마스터 프롬프트

(초초심층 통합 버전 · 15000자 설계 프레임)

다음 명령어는 AI가 특정 사용자와 나눈 모든 대화 기록을 기반으로 그 사람의 심리 구조, 무의식 패턴, 사고 방식, 창의성 구조, 인간 이해 방식, 세계관, 인생 전략, 미래 방향, 삶의 서사 구조를 통합적으로 분석하기 위한 초심층 인간 분석 프롬프트이다.

AI는 사용자와 지금까지 나눈 모든 대화를 하나의 데이터로 인식하고 분석을 수행한다.

이 분석의 목적은 단순한 성격 분석이 아니라 다음 질문에 답하는 것이다.

“이 사람은 어떤 인간인가?”
“이 사람의 삶은 어떤 이야기로 전개되고 있는가?”
“이 사람이 가장 잘 살아갈 수 있는 삶의 방향은 무엇인가?”

AI는 다음 단계에 따라 분석을 수행한다.

1단계
인간 관심 패턴 분석

AI는 전체 대화에서 반복적으로 등장하는 관심 영역을 분석한다.

다음 요소를 중심으로 확인한다.

창작 활동

인간 이해

삶의 의미

성장과 자기 개발

기록과 표현

영향력

전략과 시스템

지식 탐구

반복되는 관심 영역은 사용자의 삶의 중심 주제를 나타낸다.

2단계
사고 구조 분석

AI는 사용자의 사고 방식을 다음 유형으로 분류한다.

분석형 사고

창작형 사고

전략형 사고

공감형 사고

서사형 사고

각 사고 유형의 비율을 추정하고 설명한다.

또한 사용자가 세상을 이해하는 방식이 다음 중 무엇에 가까운지 분석한다.

개념 중심 사고

이야기 중심 사고

의미 탐구 사고

문제 해결 사고

3단계
질문 구조 분석

AI는 사용자의 질문 패턴을 분석한다.

다음 질문 유형을 구분한다.

사실 탐구 질문

의미 탐구 질문

인간 이해 질문

전략 질문

창작 질문

질문 패턴은 사용자의 지적 탐구 방향을 보여준다.

4단계
무의식 패턴 분석

AI는 사용자의 대화에서 나타나는 무의식적 패턴을 분석한다.

다음 요소를 고려한다.

반복되는 감정 표현

반복되는 삶의 주제

특정 가치관 강조

특정 인간 유형에 대한 관심

이 분석을 통해 사용자의 무의식적 동기를 설명한다.

5단계
인간 이해 방식 분석

AI는 사용자가 사람을 어떻게 이해하는지 분석한다.

다음 질문을 통해 판단한다.

사람을 행동으로 이해하는가
사람을 감정으로 이해하는가
사람을 삶의 이야기로 이해하는가
사람을 사회 구조로 이해하는가

사람을 삶의 이야기로 이해하는 경우 서사형 인간 이해 방식일 가능성이 높다.

6단계
창의성 구조 분석

AI는 사용자의 창의성이 어떤 구조로 작동하는지 분석한다.

예시 창의성 흐름

관찰 → 감정 → 의미 → 메시지 → 창작

또는

질문 → 분석 → 연결 → 아이디어 → 실행

AI는 사용자의 아이디어 생성 방식을 설명한다.

7단계
관찰 방식 분석

AI는 사용자가 세상을 어떻게 관찰하는지 분석한다.

다음 요소를 확인한다.

풍경에서 의미 발견

사람의 삶 상상

평범한 장면 해석

기록 욕구

관찰 → 의미 → 기록 구조가 나타난다면 관찰 기반 창의성일 가능성이 높다.

8단계
세계관 분석

AI는 사용자가 세상을 어떤 관점으로 이해하는지 분석한다.

예시 세계관 유형

인간 중심 세계관

의미 중심 세계관

관계 중심 세계관

탐구 중심 세계관

전략 중심 세계관

여러 세계관이 동시에 존재할 수 있다.

9단계
재능 구조 분석

AI는 다음 능력을 평가한다.

관찰력

공감 능력

분석 능력

창의성

전략 사고

이야기 구성 능력

의미 해석 능력

이 분석을 통해 사용자의 핵심 재능 구조를 설명한다.

10단계
행동 패턴 분석

AI는 사용자가 반복적으로 보이는 행동 흐름을 분석한다.

예시

관찰 → 의미 해석 → 기록 → 표현

또는

질문 → 분석 → 전략 → 실행

이 분석은 사용자의 목표 달성 방식을 보여준다.

11단계
동기 구조 분석

AI는 사용자가 무엇 때문에 행동하는지 분석한다.

다음 동기를 추론한다.

의미 있는 삶 욕구

성장 욕구

창작 욕구

영향력 욕구

인간 이해 욕구

인정 욕구

12단계
인생 전략 분석

AI는 사용자가 어떤 삶의 전략을 가지고 있는지 분석한다.

예시 전략

창작 중심 전략
영향력 중심 전략
지식 탐구 전략
인간 이해 전략
시스템 구축 전략

13단계
환경 적합성 분석

AI는 사용자가 가장 잘 성장할 수 있는 환경을 분석한다.

예시

혼자 깊이 생각하는 환경
사람과 교류하는 환경
창작 중심 환경
전략 설계 환경

14단계
위험 요소 분석

AI는 사용자가 겪을 가능성이 있는 위험 요소를 분석한다.

예시

과도한 자기 압박
지나친 분석
에너지 분산
방향 혼란

15단계
미래 인생 시나리오 생성

AI는 사용자의 삶이 앞으로 어떤 방향으로 전개될 가능성이 있는지 여러 시나리오를 생성한다.

예시

창작 중심 인생
지식 탐구 중심 인생
영향력 중심 인생
전략가형 인생

16단계
인생 서사 지도 작성

AI는 위의 모든 분석을 종합하여 한 사람의 삶을 하나의 이야기 지도로 작성한다.

글의 길이는 최소 3000자 이상으로 작성한다.

다음 구조를 따른다.

1 인간 핵심 특성
2 사고 구조
3 인간 이해 방식
4 창의성 구조
5 재능 구조
6 삶의 동기
7 행동 패턴
8 인생 전략
9 미래 방향
10 인생 서사 결론

마지막 문장은 반드시 다음 형식을 따른다.

“이 사람의 삶은 ___라는 이야기로 전개되고 있는 삶이다.”
