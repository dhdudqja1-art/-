AI 성격 + 재능 분석 명령어 (초심층 버전 · 1500자 이상)

다음 명령어는 AI가 특정 사용자와 나눈 모든 대화 기록을 기반으로 그 사람의 **성격 구조(Personality Structure)**와 **재능 구조(Talent Structure)**를 분석하기 위한 심층 분석 프롬프트이다.

AI는 사용자와 지금까지 나눈 모든 대화를 하나의 데이터로 인식하고 분석을 수행한다. 분석 과정에서는 단순히 문장의 의미를 해석하는 것이 아니라 질문 방식, 사고 흐름, 표현 스타일, 문제 접근 방식, 관심 영역, 행동 의도 등을 종합적으로 고려한다.

이 분석의 핵심 질문은 다음과 같다.

“이 사람은 어떤 성격 구조를 가진 사람이며, 어떤 재능을 가지고 있는가?”

AI는 다음 단계에 따라 분석을 수행한다.

1단계

성격 표현 패턴 분석

AI는 대화 속에서 나타나는 성격 표현을 분석한다.

다음 요소를 관찰한다.

감정 표현 방식

의견 표현 방식

질문 태도

타인에 대한 태도

자기 인식 방식

이를 통해 사용자가 내향적 / 외향적 / 탐구적 / 표현적 / 신중한 성격 중 어떤 성향이 강한지 판단한다.

2단계

인지 스타일 분석

AI는 사용자가 정보를 처리하는 방식(인지 스타일)을 분석한다.

다음 유형 중 어떤 성향이 강한지 판단한다.

직관 중심 사고
논리 중심 사고
감정 중심 사고
관찰 중심 사고
전략 중심 사고

이 분석을 통해 사용자가 세상을 이해하는 기본 인지 방식을 설명한다.

3단계

문제 접근 방식 분석

AI는 사용자가 문제를 해결할 때 어떤 접근 방식을 사용하는지 분석한다.

예를 들어 다음 질문을 통해 판단한다.

문제를 분석하려 하는가
의미를 찾으려 하는가
전략을 세우려 하는가
사람의 감정을 고려하는가
새로운 아이디어를 찾으려 하는가

이 분석을 통해 사용자의 문제 해결 스타일을 설명한다.

4단계

재능 패턴 분석

AI는 대화 속에서 나타나는 능력을 분석한다.

다음 능력을 평가한다.

언어 표현 능력
관찰력
공감 능력
논리적 사고
패턴 인식 능력
아이디어 생성 능력
시스템 이해 능력

이 분석을 통해 사용자의 핵심 재능 구조를 설명한다.

5단계

강점 성향 분석

AI는 사용자의 강점을 다음 영역에서 분석한다.

지적 강점
감정 강점
창의적 강점
전략적 강점
관찰 능력

각 영역에서 어떤 능력이 두드러지는지 설명한다.

6단계

약점 성향 분석

AI는 사용자가 겪을 가능성이 있는 약점을 분석한다.

예를 들어 다음 요소를 고려한다.

과도한 생각
결정 지연
자기 비판
에너지 분산
완벽주의

이 분석을 통해 사용자가 조심해야 할 성향을 설명한다.

7단계

성장 잠재력 분석

AI는 사용자의 성격과 재능을 기반으로 다음 질문에 답한다.

어떤 능력이 가장 크게 성장할 가능성이 있는가
어떤 활동이 재능을 가장 잘 활용할 수 있는가
어떤 환경에서 가장 빠르게 발전할 가능성이 있는가

이를 통해 사용자의 잠재 성장 방향을 설명한다.

8단계

성격 유형 종합 분석

AI는 사용자의 성격을 다음 유형 중 어떤 조합에 가까운지 설명한다.

탐구형 인간
창작형 인간
분석형 인간
공감형 인간
전략형 인간

한 사람에게 여러 유형이 동시에 나타날 수 있다.

9단계

재능 프로파일 작성

AI는 위의 모든 분석을 종합하여 사용자의 재능 프로파일을 작성한다.

다음 요소를 설명한다.

핵심 능력
보조 능력
잠재 능력
성장 가능 영역

10단계

종합 인간 프로파일 작성

AI는 위의 모든 분석을 종합하여 한 사람의 성격과 재능을 설명하는 분석 글을 작성한다.

글의 길이는 최소 1500자 이상으로 작성한다.

글의 구조는 다음 순서를 따른다.

성격 구조 설명

인지 스타일 설명

문제 해결 방식

핵심 재능

강점 성향

약점 성향

성장 잠재력

인간 유형 종합 결론

글의 톤은 심리 분석가가 한 사람의 성격과 재능을 분석하는 시선으로 작성한다.

마지막 문장은 반드시 다음 형식을 따른다.

“이 사람은 ___한 성격 구조와 ___한 재능 구조를 가진 사람이다.”
