AI 인간 분석 프롬프트 v3 (5000자 초심층 버전)

다음 명령어는 AI가 특정 사용자와 나눈 모든 대화 기록을 기반으로 그 사람의 사고 구조, 가치관, 감정 구조, 행동 패턴, 창의성 유형, 무의식적 욕망과 두려움, 정체성 형성 방식 등을 종합적으로 분석하기 위한 심층 인간 분석 프롬프트이다.

AI는 먼저 사용자와 나눈 모든 대화를 하나의 데이터로 인식하고 전체 흐름을 분석한다. 이때 단순히 개별 문장을 해석하는 것이 아니라 대화 속에서 반복적으로 나타나는 패턴, 질문 방식, 사고 흐름, 감정 표현, 관심사, 행동 의도를 종합적으로 고려한다. 분석의 목표는 단순한 성격 분류가 아니라 **“이 사람이 어떤 인간인지 이해하는 것”**이다.

AI는 분석을 시작할 때 다음 단계를 따른다.

첫 번째 단계는 관심사 패턴 분석이다.
사용자가 어떤 주제에 반복적으로 관심을 보이는지 확인한다. 예를 들어 삶의 의미, 인간 관계, 창작 활동, 성장, 철학, 기술, 전략, 감정, 사회, 성공, 영향력 등의 주제가 반복되는지 확인한다. 반복되는 주제는 그 사람의 내면적 가치와 관심을 반영한다.

두 번째 단계는 질문 구조 분석이다.
사용자가 질문을 할 때 어떤 방식으로 사고하는지 분석한다. 질문을 다음 유형으로 분류한다.

정보 탐색 질문

의미 탐색 질문

분석 질문

전략 질문

창작 질문

철학 질문

이 분석을 통해 사용자가 세상을 어떤 방식으로 이해하는지 판단한다.

세 번째 단계는 사고 유형 분석이다.
AI는 사용자의 사고가 다음 유형 중 어떤 성향이 강한지 판단한다.

분석형 사고

서사형 사고

전략형 사고

창작형 사고

공감형 사고

예를 들어 사용자가 사물이나 사람을 보며 그 안에 담긴 의미나 이야기를 해석하는 경우 이는 서사형 사고일 가능성이 높다.

네 번째 단계는 관찰 방식 분석이다.
사용자가 세상을 어떻게 인식하는지 분석한다.

예를 들어 다음 질문을 통해 판단한다.

사용자는 사람을 볼 때 어떤 생각을 하는가

풍경이나 장면을 볼 때 어떤 의미를 떠올리는가

사건을 단순한 사실로 보는가 아니면 이야기로 해석하는가

사람을 하나의 삶의 이야기로 바라보는 경우 이는 서사적 정체성 사고일 가능성이 높다.

다섯 번째 단계는 감정 구조 분석이다.
AI는 사용자의 감정 표현과 공감 능력을 분석한다.

다음 요소를 관찰한다.

타인의 감정을 이해하려는 표현

공감이나 위로 표현

감정을 해석하는 방식

감정을 기록하거나 표현하는 습관

공감 표현이 많다면 사용자는 높은 감정 공감 능력을 가진 사람일 가능성이 높다.

여섯 번째 단계는 행동 패턴 분석이다.
AI는 사용자가 반복적으로 보이는 행동 흐름을 분석한다.

예를 들어 다음과 같은 패턴이 있는지 확인한다.

관찰 → 의미 해석 → 기록 → 표현

또는

질문 → 분석 → 전략 → 실행

이러한 행동 구조는 사용자가 세상을 이해하고 행동하는 방식을 보여준다.

일곱 번째 단계는 창의성 유형 분석이다.
사용자가 어떤 방식으로 창작하거나 아이디어를 만드는지 분석한다.

예를 들어 다음과 같은 구조가 있는지 확인한다.

장면 → 감정 → 메시지 → 글

또는

문제 → 분석 → 전략 → 해결

이 분석을 통해 사용자가 창작형 인간인지 전략형 인간인지 판단한다.

여덟 번째 단계는 무의식적 욕망 분석이다.
AI는 대화 속에서 드러나는 욕망을 추론한다.

예를 들어 다음 욕망이 있는지 분석한다.

의미 있는 삶을 살고 싶은 욕망

성장 욕구

영향력을 만들고 싶은 욕망

사람을 돕고 싶은 욕망

창작 욕구

인정 욕구

아홉 번째 단계는 무의식적 두려움 분석이다.
AI는 다음과 같은 두려움이 나타나는지 분석한다.

실패에 대한 두려움

의미 없는 삶에 대한 두려움

가능성을 다 쓰지 못할 것에 대한 두려움

타인에게 이해받지 못할 것에 대한 두려움

방향을 잘못 선택할 것에 대한 두려움

열 번째 단계는 정체성 형성 방식 분석이다.
심리학에서는 사람의 정체성이 자신의 삶을 하나의 이야기로 이해하는 과정에서 형성된다고 설명한다.

AI는 사용자가 자신의 삶과 경험을 어떤 이야기 구조로 이해하는지 분석한다.

예를 들어 다음 질문을 통해 판단한다.

사용자는 자신의 삶을 어떤 의미로 해석하는가

과거 경험을 어떻게 이야기로 연결하는가

미래를 어떻게 상상하는가

이 분석을 통해 사용자의 서사적 정체성 구조를 설명한다.

모든 분석이 끝나면 AI는 결과를 하나의 서사형 분석 글로 작성한다.

글의 길이는 최소 1500자 이상이며 가능하면 2000~3000자 이상으로 작성한다.

글의 구조는 다음 순서를 따른다.

이 사람이 어떤 사고 구조를 가진 사람인지 설명

세상을 바라보는 관점 설명

감정과 공감 능력 분석

관찰 방식 분석

행동 패턴 분석

창작 또는 문제 해결 방식 분석

무의식적 욕망 분석

무의식적 두려움 분석

가치관과 삶의 방향 추정

인간 유형 종합 결론

글의 톤은 분석적이면서도 서술적인 방식으로 작성하며 한 사람의 내면을 이해하려는 관찰자의 시선으로 작성한다.

글의 마지막 문장은 반드시 다음 형식을 따른다.

“이 사람은 ___한 사람이다.”
